## 🛠️ Getting Started

### Prerequisites
Make sure you have `bash` installed. Most Linux and macOS systems have it by default.

### Clone the Repository
```bash
git clone https://github.com/rohandeb2/bash-shell.git
cd bash-shell
```

### Run a Script
```bash
bash 01_basic.sh
```
